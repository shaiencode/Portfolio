## Features
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop views using CSS Clamp and Media Queries.
- **Dynamic Projects:** Custom media renderer for images, .mp4/.mov videos, and interactive Figma embeds.
- **Animations:** Smooth entrance effects powered by Framer Motion.
- **Form Validation:** React-based contact form validation to ensure data integrity.
- **Modern UI:** Consistent branding with a dark theme and signature accent colors.

## Tech Stack
- **Frontend:** React.js, Vite
- **Styling:** Vanilla CSS
- **Animations:** Framer Motion
- **Deployment:** GitHub Pages
